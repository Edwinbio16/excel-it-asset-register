# Excel IT Asset Management System

An Excel-based IT asset register designed to simulate real-world asset tracking in an organisation.

## Features
- Asset IDs and serial numbers
- Assigned users and locations
- Status tracking (In Use, Spare, Faulty, Retired)
- Purchase and warranty date tracking
- Notes for maintenance and lifecycle management

## Why this project
This project demonstrates structured data management, attention to detail, and how Excel can be used to support IT and administrative operations.
